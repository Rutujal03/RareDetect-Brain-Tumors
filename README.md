# RareDetect-Brain-Tumors
## Features
 - Automated brain tumor detection from MRI scans
 - Image preprocessing using resizing, normalization, and augmentation
 - Comparative analysis of VGG16, ResNet50, and Custom CNN
 - Model training and evaluation using TensorFlow and Keras
 - Interactive web application built with Streamlit
 - Performance visualization using Matplotlib and OpenCV


## Technologies Used
- Python
- TensorFlow & Keras
- OpenCV
- NumPy
- Matplotlib
- Streamlit

  ## Objective
The primary objective of this project is to assist healthcare professionals by providing an AI-powered tool for early and accurate brain tumor detection, reducing diagnostic time and improving decision-making.

## Future Enhancements
- Multi-class brain tumor classification
- Tumor segmentation and localization
- Explainable AI (XAI) integration
- Real-time clinical deployment support
